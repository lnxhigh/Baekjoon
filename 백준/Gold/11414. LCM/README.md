### 문제 설명

<p>You are given two natural numbers A and B. Determine the natural number N such that the least common multiple of the numbers A + N and B + N is minimal.</p>

### 입력 

 <p>The only line of the input contains two natural numbers: A and B. None of them is exceeding 10<sup>9</sup>.</p>

### 출력 

 <p>Output the natural number N such that LCM(A + N, B + N) is minimal. If there are several values of N which yield the minimum, output the smallest one.</p>
